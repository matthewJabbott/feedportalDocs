What is the Feed Formulation Portal?

Quick Overview: The Feed Formulation Portal is a modern web application that helps users create, manage, and optimize animal feed recipes. It's like a digital cookbook for animal nutrition, where users can manage ingredients, create custom feeds for different animals, and ensure nutritional requirements are met.

The Feed Formulation Portal is a comprehensive web-based application designed to streamline and modernize feed formulation management for real-world use cases. Built as a capstone project for Deakin University by the Gopher Industries team, it provides an intuitive interface for users to manage ingredients, feeds, and species with robust authentication, validation, and business logic suitable for production environments.

Technology Stack

The portal follows modern web development practices using the MERN stack architecture:

Frontend

React.js for interactive user interfaces

Backend

Node.js with Express.js for server-side logic

Database

MySQL for reliable data storage

Authentication

Clerk.js for secure user management

Project Purpose

This portal serves multiple important purposes:

  • Educational Foundation: Provides a comprehensive reference for future student cohorts working on similar projects
  • Industry Application: Demonstrates real-world software development practices and modern web technologies
  • Scalable Architecture: Built with production-ready patterns that can be extended and maintained
  • Collaborative Learning: Showcases effective team development workflows using GitHub, Microsoft Teams, and project management tools

Who Is This For?

Target Users:

  • Animal Nutritionists: Professionals who need to create balanced feed formulations
  • Feed Manufacturers: Companies that produce commercial animal feeds
  • Farmers and Ranchers: Individuals who want to optimize their animal feeding programs
  • Students and Researchers: Those studying animal nutrition and feed science
  • Future Developers: Students who will continue developing and maintaining this system
